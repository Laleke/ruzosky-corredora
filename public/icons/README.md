# Íconos PWA

Faltan los archivos de ícono referenciados en `public/manifest.webmanifest`:

- `icon-192.png` (192×192)
- `icon-512.png` (512×512)
- `icon-maskable-512.png` (512×512, con padding seguro para máscara)

Generarlos a partir del logo de RZK Prop antes del primer deploy.
Sin ellos, la instalación PWA en Android/iOS no mostrará ícono correcto.
